    Local Media Calls

    Local media calls are those calls in which the bot manages the call's video and audio sockets. The bots send and receive raw audio and video data over the network.

    Building a StatefulClient with Media Platform Settings

    1. Create the an instance of the IStatefulClientBuilder.
    2. Create an instance of MediaPlatformSettings and use StatefulClientExtension.SetMediaPlatformSettings method to pass the MediaPlatformSettings to build the IStatefulClient
    3. Once the builder builds the IStatefulClient, a media session can be created using either StatefulClientExtension.CreateMediaSession or StatefulClientExtension.CreateMediaSession depending on how many video sockets are required.
    4. When creating an object of the Call resource, MediaConfig needs to be set to AppHostedMediaConfig.
    Important

    When creating a new call that uses local media, an instance of IMediaSession needs to be passed into CallCollectionExtensions.AddAsync
