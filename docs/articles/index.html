    Graph Core and Stateful SDK Concepts

    This article describes the key concepts in order to effectively utilize the Graph Core SDK and the Graph Stateful SDK including the Calling SDK.

    Graph Core SDK

    The DefaultContainerClient object is the entry point to the Graph Core SDK. The majority of the Graph Core SDK is automatically generated by the Graph SDK generator using the OData resource model. It reflects the REST-ful wire protocol provided by the backend service.

    Graph Stateful SDK and Stateful Client Builder

    The IStatefulClient object is the entry point to the Graph Stateful SDK and the Calling SDK. This SDK is designed for stateful services and provides additional support on top of the Graph Core SDK for resource state and media management. The IStatefulClientBuilder is the object used to construct a new IStatefulClient with the desired settings.

    For more details on how to configure your bot please review the Setting up stateful bots with the Graph Calling SDK article.
