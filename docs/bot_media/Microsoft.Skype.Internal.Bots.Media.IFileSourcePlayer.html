    Skip(Int32)

    The number of milliseconds to skip. The value can be positive (to move forward) or negative (to move backward). This method can be called in any state. If user skips beyond the end of the file, Player will stop and event FileSourcePlayerEndOfFile will be raised. If user skips before the beginning of the file, Player will start playing from the beginning.
